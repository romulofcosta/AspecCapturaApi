v0.8.0 - Correção de CORS e adição de versionamento

## 🔧 Correções

### CORS
- **Configuração corrigida** para aceitar domínio principal do Cloudflare Pages
  - Adicionado suporte explícito a `https://pwa-camera-poc-blazor.pages.dev`
  - Mantido suporte a subdomínios (`.pwa-camera-poc-blazor.pages.dev`)
  - Corrige erro: "Access-Control-Allow-Origin header is not present"

**Antes:**
```csharp
return origin.EndsWith(".pwa-camera-poc-blazor.pages.dev");
```

**Depois:**
```csharp
return origin == "https://pwa-camera-poc-blazor.pages.dev" ||
       origin.EndsWith(".pwa-camera-poc-blazor.pages.dev");
```
